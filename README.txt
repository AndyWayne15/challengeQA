Instrucciones de Ejecución:
- Serenity BDD: 3.6.12
- Java: 1.8
- Gradle 7.6.1

Pasos para Ejecutar el Programa:
1. Clonar o descargar el repositorio desde GitHub.
2. Abri el proyecto en IntelliJ
3. Instalar las dependencias utilizando Gradle: ./gradlew clean build
4. Ejecutar las pruebas automatizadas: ./gradlew test
5. Revisra los reportes generados en /target/site/serenity/index.html

Nota: Asegúrarse de tener configurado el entorno con las versiones especificadas de las tecnologías mencionadas.
Como el ambiente JAVA_HOME y Gradle en el entorno de Windows.
